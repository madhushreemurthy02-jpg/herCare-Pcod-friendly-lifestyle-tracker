# herCare 🌸
### Your Personal PCOD Wellness Companion

herCare is a responsive health and wellness web application designed 
specifically for women managing PCOD (Polycystic Ovary Disease). 
It helps users track their menstrual cycle, log daily health habits, 
monitor symptoms, and receive AI-powered personalised wellness insights 
— all in one gentle, beautifully designed platform.

## ✨ Features
- 🔐 User Authentication — Secure login & signup
- 👤 Health Profile Setup — Age, height, weight & cycle length
- 📓 Daily Log — Sleep, hydration, mood, nutrition & activity tracking
- 🩸 Cycle Tracker — Period logging, symptom tracking, flow & pain monitoring
- 🤖 AI Insights — Gemini AI-powered personalised wellness analysis
- 🔔 Gentle Reminders — Customisable daily wellness alerts


## 🛠️ Tech Stack
### Frontend
- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Dancing Script, Nunito)

### Backend
- Python 3.10+
- Flask & Flask-JWT-Extended
- MongoDB (PyMongo)
- Flask-Bcrypt for password hashing
- REST API architecture

## 🚀 Getting Started
1. Clone the repo
2. Open frontend with VS Code Live Server
3. Set up backend with Python + MongoDB
4. Add your Gemini API key
5. Run and enjoy! 🌷
